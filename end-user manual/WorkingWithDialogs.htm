Working with Dialogs
 
OK
Press the OK button to accept all changes that you have made in the dialog. For a dialog with multiple tabs, the OK button will save changes on all tabs.
Cancel
Press the Cancel button to close the dialog without making any changes.
Apply

Press the Apply button to apply all changes, but keep the dialog open. Note that the applied changes can not be undone, even if you press the Cancel button afterwards.

Close
Press the Close button to close the dialog.
